Welcome to the G-Blog API. Below is the list of available endpoints.

üìù Blog APIs

POST /postblog/

Description: Create a new blog post.

Request (form-data):
  title: string
  content: string
  image: file
  user_id: int
Response:
  { "message": "Blog post created successfully", "post": {...} }

GET /get_blog_posts/

Description: Get all blog posts.

GET /get_blog_post/<post_id>/

Description: Get a single blog post by ID.

üîë Authentication

POST /signin/

Request (JSON):
  { "email": "user@example.com", "password": "secret" }
Response:
  { "message": "Successfully logged in", "user_id": 1 }

POST /signup/

Request (form-data):
  name, phonenumber, email, password, avatar (file)
Response:
  { "message": "Successfully signed up" }

GET /resetpassword/<email>/

Description: Send password reset email.

GET /logout/

Description: Logout user.

üí¨ Messaging

POST /send_message/

Request (JSON):
  { "user_id": 1, "message": "Hello!" }
Response:
  { "message": "Message sent successfully", "message_id": 2 }

üì¶ Orders

POST /create_order/

Request (JSON):
  { "user_id": 1, "product_name": "Laptop", "quantity": 2, "price": 1200 }
Response:
  { "message": "Order created successfully", "order_id": 5 }

POST /create_product_order/

Description: Place an order for a product.

POST /create_bulk_order/

Description: Place multiple product orders in one request.

GET /get_product_orders/<user_id>/

Description: Get pending orders of a user.

GET /get_all_orders/

Description: Get all orders (admin).

GET /confirm_order/<order_id>/

Description: Mark an order as delivered.
